What is Startup Staff?

Startup Staff is an agentic AI-powered solution designed to give founders instant access to a cross-functional core team. Rather than consulting a single AI, your query is routed through a team of six specialist personas — each bringing a distinct professional lens — before Soleil, your Chief of Staff, synthesises a unified strategic response.

How to Use

1
Configure your API
Click ⚙ API Config, select a provider (GMI, Claude, ChatGPT, DeepSeek, or Mistral), enter your API key, and select a model.
2
Set your Startup Profile
Click ✏ Profile to enter your name and a brief description of your startup. This context is injected into every query.
3
Ask anything
Type a strategic question in the Soleil Interface. The team will consult, deliberate, and respond with a synthesised recommendation.
4
Generate documents
Prefix your request with "generate a document:", "create a spreadsheet:", or "build an HTML page:" to produce downloadable files.
5
Add context
Use ⊞ Knowledge Base to upload internal documents or reference URLs the team will consult before responding.
